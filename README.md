# Steam Games — Hit Prediction

Data analysis and ML project based on the [Steam Games Dataset](https://www.kaggle.com/datasets/fronkongames/steam-games-dataset).

## Goal
Predict whether a game will be a hit, defined as having at least 1,000 positive reviews and a positive rating of 80% or above. Three classifiers were compared: Logistic Regression (baseline), Random Forest, and XGBoost.
